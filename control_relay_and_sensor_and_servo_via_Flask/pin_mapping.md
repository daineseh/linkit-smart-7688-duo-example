# Pin Mapping
```
Relay pin | MT7688 pin
======================= 
Vcc       | 5V
GND       | GND
SIGNAL    | D14
```

```
Sensor pin | MT7688 pin
======================= 
Vcc        | 5V
GND        | GND
SIGNAL     | D3
```

```
Servo pin | MT7688 pin
======================= 
Vcc        | 5V
GND        | GND
SIGNAL     | D5
```
